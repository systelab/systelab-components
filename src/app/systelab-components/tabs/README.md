# systelab-tabs

Component to show more than one panel.

## Using the template

```html
<systelab-tabs class="slab-flex-1">
  <systelab-tab class="slab-flex-1" [title]="'Tab 1'">
    Tab 1 content
  </systelab-tab>
  <systelab-tab class="slab-flex-1" [title]="'Tab 2'">
    Tab 2 content
  </systelab-tab>
</systelab-tabs>
```
